# PLAN TEMPLATE — ideas-tab / Markets workstream

> **Who this is for:** an implementation agent (assume a strong but HASTY model) executing
> under `.claude/skills/executing-plans`. The plan is written so that agent needs **zero
> improvisation**: every step carries complete code, the exact verification command, and the
> expected output. The executing agent's contract: **STOP on any mismatch between the plan
> and the tree; a step is DONE only when its verification command's actual OUTPUT is pasted
> and that output proves the behavior fired.** "It passes" closed the WHIPPYX vacuous test
> as green while it verified NOTHING — never again.
>
> **How to author a plan from this template:** fill every section; delete none. Every code
> block is a full replacement block (old text → new text), not a sketch — sketches are where
> hasty models improvise. Every line number you cite must be true at the HEAD you name
> (grep it the day you write; the Wave-11 agent narrated code that contradicted the diff,
> so narration is never a source). Every step gets a HASTY-MODEL TRAP naming the specific
> mistake an eager model makes on that step.
